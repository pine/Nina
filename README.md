## Requirements

- Node v`$(cat .node-version)`
- Yarn

## Running Locally

```
$ git clone https://github.com/pine/Nina.git
$ cd Nina
$ yarn
$ yarn start
```

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
